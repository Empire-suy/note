## 坑点

1. 无法动态绑定class
2. 无法动态绑定v-show，可以动态绑定v-if【可能是层级嵌套的问题导致的】
3. 无法穿直接穿透类名使用？需要给组件添加一个类名，然后通过deep的方式选中当前的类名再给目标元素设置样式
4. ref无法获取到内置组件的实例

  ```ts
  const {ctx} = getCurrentInstance();

  const open = () => {
    // 通过获取当前实例再获取到最新的值
    // 无法获取到内置组件的实例
    ctx.$refs.ref.open();
  }
  ```
