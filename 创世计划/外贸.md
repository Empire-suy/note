### 数据库设计

User

| 字段名               | 数据类型                                | 约束与说明                                       |
| :------------------- | :-------------------------------------- | :----------------------------------------------- |
| `id`                 | BIGINT                                  | 主键, 自增或随机生成，唯一标识每个用户           |
| `username`           | VARCHAR(50)                             | 非空, 唯一, 用户名                               |
| `email`              | VARCHAR(255)                            | 非空, 唯一, 电子邮件地址                         |
| `password_hash`      | VARCHAR(255)                            | 非空, 存储加密后的密码哈希值                     |
| `created_at`         | TIMESTAMP                               | 默认当前时间戳, 记录账户创建的时间               |
| `updated_at`         | TIMESTAMP                               | 默认当前时间戳, 自动更新, 修改时更新时间         |
| `full_name`          | VARCHAR(100)                            | 完整的名字, 可选字段                             |
| `profile_image_url`  | TEXT                                    | 用户头像链接, 可选字段                           |
| `bio`                | TEXT                                    | 简短自我介绍, 可选字段                           |
| `status`             | ENUM('active', 'inactive', 'suspended') | 默认 'active', 用户状态                          |
| `last_login_at`      | TIMESTAMP                               | 最后一次登录的时间, 可选字段                     |
| `failed_attempts`    | TINYINT                                 | 默认 0, 登录失败次数                             |
| `lockout_expires_at` | TIMESTAMP                               | 账户锁定时记录解锁时间, 可选字段                 |
| `role_id`            | BIGINT                                  | 外键关联到角色表                                 |
| `group_ids`          | JSON 或 ARRAY                           | 用户所属组别的集合, 适用于多对多关系             |
| `preferences`        | JSON                                    | 用户偏好设置, 比如语言选择、主题模式             |
| `social_media_links` | JSON                                    | 包含社交媒体账号链接的信息, 如 Twitter, Facebook |
| `is_verified`        | BOOLEAN                                 | 默认 false, 标记用户是否已验证身份               |
